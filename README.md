# JARVIS (OpenClawd App)

Desktop app (Electron + React + SQLite) with:
- JARVIS dashboard UI
- Task Board for OpenClawd workflow
- Audit Log viewer + controls
- Telegram and VPS sync settings

## 1) Prerequisites (Windows)

Install these first:
1. **Git for Windows** (includes Git Bash)
2. **Node.js 20 LTS** (or newer LTS)
3. **Visual Studio Build Tools 2022** with:
   - Desktop development with C++
   - MSVC toolset
   - Windows 10/11 SDK

> Why Build Tools? `better-sqlite3` is a native module used by this app.

Verify installs:

```bash
git --version
node -v
npm -v
```

## 2) Clone and install

```bash
git clone <YOUR_GITHUB_REPO_URL>
cd OPENCLAWD-APP/app
npm install
```

## 3) Run in development (recommended)

Open **two terminals**:

### Terminal A (Vite UI)
```bash
cd OPENCLAWD-APP/app
npm run dev
```

### Terminal B (Electron shell)
```bash
cd OPENCLAWD-APP/app
npm run electron:dev
```

The app should open as a desktop window titled **JARVIS**.

## 4) Production build (optional)

```bash
cd OPENCLAWD-APP/app
npm run electron:build
```

Packaged output will be generated by `electron-builder`.

## 5) Where local data is stored

The app database path is:
- `%APPDATA%\BobAssistant\bob.db`

If you want to reset local data, close the app and delete that file.

## 6) First-time in-app setup checklist

1. Open **Settings** tab.
2. Set **Theme**.
3. Open **Audit Log** card to verify logs are being recorded.
4. Optional: Configure Telegram token and test send.
5. Optional: Configure VPS Sync URL and test sync.
6. Open **Task Board** and create your first task.

## 7) Configure GitHub push from your computer

Inside your local repo:

```bash
git remote add origin https://github.com/<user>/<repo>.git
# if origin already exists, use:
# git remote set-url origin https://github.com/<user>/<repo>.git

git push -u origin work
```

For HTTPS auth, use a GitHub **Personal Access Token (PAT)** as password.

## 8) Common issues

### `better-sqlite3` install fails on Windows
- Make sure Visual Studio Build Tools + C++ workload are installed.
- Re-run:
  ```bash
  npm rebuild better-sqlite3
  npm install
  ```

### Electron window opens but UI is blank
- Confirm Vite is running in Terminal A.
- Check Terminal B output for startup errors.

### Port 5173 is busy
- Stop the process using it or run Vite on another port and update `VITE_DEV_SERVER_URL` accordingly.
